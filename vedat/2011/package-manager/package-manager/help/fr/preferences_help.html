Préférences

Cette fenêtre est utilisée afin de définir l'intervalle entre chaque vérification de mise à jour et les paramêtres liés aux dépôts.

Dépôt

Dans cet onglet, vous pouvez ajouter, éditer ou supprimer un dépôt en utilisant les boutons ad-hoc.

Paramètres généraux

Ici, vous pouver établir les paramètres généraux pour le gestionnaire de paquets de Pardus, comme l'affichage dans la boîte à icônes, l'activation ou la désactivation de la vérification périodique de mises à jour etc.

Cache

Ici, vous pouvez choisir la taille du cache disque utilisable pour les logiciels téléchargés. Cliquer sur le bouton "Vider le cache" pour supprimer tous les paquets téléchargés et obtenir de l'espace disque. Notez que cette opération n'affectera en rien la stabilité de votre système ni ne supprimera un logiciel installé.
