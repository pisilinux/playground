Paket-Manager

Paket-Manager wird für die Suche, Installation oder für Upgrades der Pakete vom Pardus-Paket-Depo. Es ist ein handliches und brauchbares Tool, um Ihre Pakete einfach zu verwalten.

Installation einer neuen Software

Um ein Paket neu zu installieren, klicken Sie auf "Neue Pakete anzeigen" und finden the Software, an der Sie interessiert sind. Klicken Sie auf Paket(e) installieren, die Pakete und Ihre Abhängigkeiten, die auch installiert werden müssen, werden angezeigt.

Installationsvorgang wird von Ihrer Bandbreite und Internet-Zugang abhängig sein. Sie können den Fortschritt während der Installation der Pakete sehen.

Entfernen eines Pakets

Um ein Paket zu entfernen, klicken Sie auf den Button "Installierte Pakete anzeigen". Klicken Sie auf das Paket, welches Sie entfernen möchten und klicken anschließend auf den Button "Paket(e) entfernen" Die Pakete, die Sie gewählt haben, werden vom System entfernt.

Updates anzeigen

By default, you'll see the package updates on the system tray. Wenn Sie den Paket-Manager öffnen und auf "Updates anzeigen" klicken, können Sie die verfügbaren System-Updates sehen. Klicken Sie auf "Paket(e) upgraden" um Ihr System zu aktualisieren.

Paket-Suche

Um ein Paket zu suchen, geben Sie einfach den Namen des Pakets in die Suchleiste ein. Ihnen werden eine Liste von Paketen aufgelistet, die mit Ihrer Suche übereinstimmen.
