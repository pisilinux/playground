Configuración

En esta ventana se especifíca el período de actualización y los repositorios.

Repositorios

En esta pestaña puede agregar, modificar o remover repositorios.

Configuraciones generales

Aqui se puede indicar la configuración general del administrador de paquetes de Pardus, p.ej. mostrar ícono en barra de tareas, habilitar/deshabilitar chequeo de paquetes actualizados, etc.

Cache

Aqui puede elegir cuanto espacio en disco usará para el software descargado. Al presionar "Vaciar Cache" se borrarán todos los paquetes descargados anteriormente y liberará un poco de espacio en disco. Note que esto no afectará la estabilidad del sistema, tampoco se borrará software instalado con esta operación.
