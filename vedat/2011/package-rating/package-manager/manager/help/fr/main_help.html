Gestionnaire de Paquets

Le Gestionnaire de Paquet sert à chercher, installer ou mettre à jour des paquets à partir du dépôt de paquets Pardus. C'est un outil très pratique et ergonomique pour gérer vos paquets de manière simple.

Installer un nouveau logiciel

Afin d'installer un nouveau paquet, cliquer sur l'onglet "Montrer les nouveaux paquets" et trouvez les fichiers qui vous intéressent. Cliquer sur l'onglet "Installer les paquers" provoque l'affichage des paquets à installer ainsi que leurs dépendances.

Le temps d'installation dépend de votre bande passante et de votre accès Internet. Vous pourrez voir la progression au fur et à mesure de l'installation

Supprimer un paquet

Afin de supprimer un paquet, cliquez sur le bouton "Lister les paquets installé". Cliquez sur le paquet que vous voulez supprimer puis cliquez sur le bouton "Supprimer les paquets". Les paquets sélectionnés seront alors supprimés du système.

Montrer les mises à jour

Par défaut, vous verez les mises à jour de paquets dans la boîte à icônes. Après avoir lancé le gestionnaire de paquets, cliquer sur "Montrer les mises à jour" pour voir les mises à jour du système disponibles. Cliquer sur "Mettre à jour les paquets" afin de mettre à jour votre système.

Recherche de paquet

Pour rechercher un paquet, ecrire simplement son nom dans la barre de recherche. Une liste de paquets qui correspondent à votre recherche vous sera alors présentée.
