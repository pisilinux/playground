Cache

Hier kun je kiezen hoeveel schijfcache gebruikt mag worden voor gewonloade software. Een klik op de "Opruimen schijfcache"-knop wist alle gedownloade pakketten en geeft zodoende wat schijfruimte terug. Dit zal de stabiliteit van het systeem niet beinvloeden, en ook de geinstalleerde software niet verwijderen.
