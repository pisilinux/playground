Pakethanteraren

Pakethanteraren används till att söka, installera och uppdatera paket från Pardus programförråd. Det är ett användbart verktyg för att hantera paket på ett enkelt sätt.

Installera nya paket

För att installera ett nytt paket, klicka på "Visa nya paket" och välj de program du är intresserad av. Klicka sedan på "Installera paket" så visas de paket som ska installeras och deras eventuella beroenden.

Installationstiden beror av din internetanslutning. Du kommer att se hur installationen fortskrider.

Ta bort paket

För att ta bort paket, klicka på "Visa installerade program". Välj de paket du vill ta bort, klicka sedan på "Ta bort paket". De paket du har valt kommer att avinstalleras från ditt system.

Visa uppdateringar

Som standard visas paketuppdateringar i systembrickan. Om du öppnar Pakethanteraren, klicka på "Visa uppdateringar" för att se tillgängliga uppdateringar. Klicka på "Uppdatera paket" för att uppdatera systemet.

Paketsökning

Skriv ett paketnamn i sökfältet för att söka efter ett paket. En paketlista som matchar din sökning kommer att visas.
